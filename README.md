Dependencies
===
* [Node.js](http://nodejs.org/)
* Browserify
```Bash
npm install -g browserify
```
* Reactify
```Bash
npm install -g reactify
```
* Python libs
```Bash
pip install -r requirements.txt
```
* MySql
```Bash
apt-get install mysql-server
```
* MongoDb
```Bash
apt-get install mongodb
```
